HyperProc Help

?, help Displays this help
load load an image in current buffer
configloads a quantizer configuration file. Default: default.xml
exec executes a script
save save current image to disk
reload reloads last loaded image into buffer
fwt Does the Wavelet Tranform
iwt Does the Inverse Wavelet Transform
dct Does the Discrete Cosinus Tranform
idct Does the Inverse Discrete Cosinus Transform
quant Quantizes the image
dequant Dequantizes the image
label labels a quantized image and saves it to disk. Default: default.lab
labdecode loads a labeled image and displays the quantized image. Default: default.lab
indexcode applies Index Coding algorithm and saves it to disk. Default: default.coded
indexdecode decodes an image from disk. Default: default.coded
byte changes datatype to byte
int changes datatype to integer
float changes datatype to float
mult multiplies a constant
diff computes the difference between shown and last loaded image.
blur Blurs the image
sharpen Sharpens the borders
medianf Applies the median filter to supress noise
gradmag Applies the gradient magnitude
show/hide When hidden, the result of applying an operator is not shown
size prints the image size
type prints the image type
retile retiles an image (creates a new instance)
tileset changes tile dimension
level prints or modifies number of levels of the wavelet
visualization changes the way to display current image: absolute, rescale, inverse
close, quit Exits the program
